# findipsrv

Find IP Service

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'findipsrv', github: 'frostdot/findipsrv'
```

And then execute:

    $ bundle install

## Usage

Starting service:

    # findipsrv

Client search command:

    $ findip
